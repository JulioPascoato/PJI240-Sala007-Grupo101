Acessibilidade deste site

A interface do Acervo Digital da kolombolo-di-piratininga foi construída pensando em facilitar o acesso de todos os usuários, incluindo aqueles que possuem necessidades especiais.

Navegação via teclado

É possível realizar uma navegação mais rápida pelo site utilizando as seguintes teclas de atalho:

  1. Teclando-se ALT + 1, chega-se ao página principal, onde é possível navegar pelas funcionalidades principais
  2. Teclando-se ALT + 2, chega-se ao campo de busca geral, onde será possível definir os parâmetros para busca de materiais;
  3. Teclando-se ALT + 9 exibirá esta página listando os recursos de acessibilidade do site.

As teclas citadas anteriormente funcionam corretamente nos navegadores Google Chrome e Internet Explorer.

No caso do navegador Firefox, ao invés de ALT + número, deve-se teclar simultaneamente ALT + SHIFT + número.

No navegador Opera, as teclas são SHIFT + ESCAPE + número. Teclando SHIFT + ESCAPE, uma janela é aberta com todas as alternativas de ACESSKEY da página. No Firefox e no Opera, ao teclar utilizando suas alternativas de acesso, o usuário vai diretamente para a página proposta; enquanto que no Internet Explorer, basta posicionar o mouse no link para clicar, e, no caso da navegação via teclado, é preciso pressionar ENTER para acessar a página desejada.

Descrição da Estrutura do Site

O Acervo Digital da kolombolo-di-piratininga está estruturado da seguinte maneira:

Cabeçalho

No topo de cada página existem os seguintes elementos:

  1. Logo da Kolombolo: o logo possuí um link direcionando à página principal do Acervo Digital;
  2. Menus diversos: links para para navegação e utilização do sistema;
  3. Barra de busca: barra com um campo de entrada para valores utilizados na busca de materiais.

Conteúdo central

Esta é a parte do site responsável pela apresentação do conteúdo principal de cada página.

Rodapé

No rodapé são apresentados basicamente os logos das nossas mídias sociais.

Realização de buscas

O Acervo Digital fornece mecanismos de busca para facilitar a localização dos materiais desejados. Exitem campos de busca no cabeçalho de cada página (utilizado para buscas completas) e busca refinadas durante a navegação (apresentada nas páginas de comunidades e coleções). O resultado de cada busca é apresentado ao usuário na forma de tabelas.
